Contact API

List Contact


{{contactLink}}?name={name}&number={number}

Parameter Description
name Contact name.
number Phone number.

Response

{"contacts":[{"emails":[],"groups":[{"id":"7","name":"My Contacts"}],
"name":"Contact1","vCard":"BEGIN:VCARD\r\nVERSION:2.1\r\nN:;Contact1;;;
\r\nFN:John Smith\r\nTEL;CELL:0123456789\r\nEND:VCARD\r\n",
"phones":[{"displayLabel":"Mobile","number":"0123456789","type":2}],"personId":1116}],
"description":"","requestMethod":"GET","isSuccessful":true}

Field Description
contacts A list of contact objects with the following fields

emails
List of emails.
groups
List of groups.
name
Contact name.
vCard
Contact vCard
phones
List of phones.
personId
Unique identifier.
ringTone
Ringtone content URI.
description Contains the error message if isSuccessful is "false".
requestMethod Request method. Default to "GET", "POST" or "DELETE" depending on the HTTP method.
isSuccessful true if operation is successful. Otherwise returns false.

Add Contact


{{contactLink}}

Parameter Description

mobile Mobile phone number.
home Home phone number.

jobtitle Job title.

Delete Contact


{{contactLink}}{id}

Parameter Description
id Person id which is a unique identifier for the contact.

Response

{ "count": 1, "description": "Successfully deleted the contact with id 35976",
"requestMethod": "DELETE", "isSuccessful": true }

Field Description
count Number of deleted contact. Should be 1 if the deletion is successful.
description Contains the error message if isSuccessful is "false".
requestMethod Request method. Default to "GET", "POST" or "DELETE" depending on the HTTP method.
isSuccessful true if operation is successful. Otherwise returns false.
