USSD API

In order for USSD APIs to work, myMobKit accessibility service must be turned on under Settings->Accessibility.

Send USSD


{{ussdLink}}

Parameter Description
Command USSD command to send. E.g. *123#.
Pattern Expected response pattern from the command. Provide this value if the response pattern is consistent, e.g. if the response always starts with "Your current balance is ...".

Response

{ "responsePattern": "", "sessionId": "329797043932645619", "ussdCommand": "*128%23",
"description": "", "requestMethod": "POST", "isSuccessful": true }

Field Description
responsePattern The expected response pattern.
sessionId Session id which will be used to query the response later.
ussdCommand The USSD command.
description Contains the error message if isSuccessful is "false".
requestMethod Request method. Default to "GET", "POST" or "DELETE" depending on the HTTP method.
isSuccessful true if operation is successful. Otherwise returns false.
