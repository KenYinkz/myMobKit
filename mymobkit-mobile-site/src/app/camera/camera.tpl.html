Camera API


Capture Photo


{{cameraLink}}?flash={0|1}&front={0|1}&resolution={0|1|2|3}&focus={0|1|2|3|4|5|6}

Parameter Description
flash Optional parameter. 0 for no flash, any other values indicate flash is required. Default to 0.
front Optional parameter. 0 for rear facing camera, any other values indicate front facing camera. Default to 0.
resolution Optional parameter. Resolution to be used to capture the photo. Possible values are
0
Default resolution.
1
Low resolution.
2
Medium resolution.
3
High resolution.
focus Optional parameter. Focus mode to be used to capture the photo. Possible values are
0
Auto-focus mode.
1
Continuous auto focus mode intended for taking pictures.
2
Continuous auto focus mode intended for video recording.
3
Extended depth of field (EDOF). Focusing is done digitally and continuously.
4
Focus is fixed.
5
Focus is set at infinity.
6
Macro (close-up) focus mode.

Response

Photo in JPEG format.
