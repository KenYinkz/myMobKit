Call API


Make Call


{{callLink}}

Parameter Description
Destination Mandatory parameter. The contact name or phone number to call.

Response

{ "destination": "+60123456789", "description": "",
"requestMethod": "POST", "isSuccessful": true }

Field Description
destination Destination number to call.
description Contains the error message if isSuccessful is "false".
requestMethod Request method. Default to "GET", "POST" or "DELETE" depending on the HTTP method.
isSuccessful true if operation is successful. Otherwise returns false.

Response

{ "callHistories": [
{ "callDuration": "7", "callTime": "2015-10-19T20:08:28GMT+08:00",
"callType": "1", "phoneNumber": "0126868739" },
{ "callDuration": "43", "callTime": "2015-10-19T23:35:56GMT+08:00",
"callType": "2", "phoneNumber": "0320535153" },
{ "callDuration": "0", "callTime": "2015-10-20T11:15:07GMT+08:00", "callType": "3",
"phoneNumber": "0326015000" } ],
"description": "", "requestMethod": "GET", "isSuccessful": true }

Field Description
callHistories Call histories
callDuration
The duration of the call in seconds.
callTime
The date the call occured.
callType
The type of the call (incoming, outgoing or missed).
Incoming - 1, Outgoing - 2, Missed - 3
phoneNumber
The phone number as the user entered it.
description Contains the error message if isSuccessful is "false".
requestMethod Request method. Default to "GET", "POST" or "DELETE" depending on the HTTP method.
isSuccessful true if operation is successful. Otherwise returns false.
