Delete Media


  {{allMediaLink}}image/{id}



  {{allMediaLink}}video/{id}



  {{allMediaLink}}audio/{id}

Parameter Description
id Media id to delete. Compulsory value which must be provided.

Response

{ "count": 1, "description": "Successfully deleted the media with id 2176",
"requestMethod": "DELETE", "isSuccessful": true }

Field Description
count Number of deleted media.

Stream Media


  {{streamMediaLink}}image?uri={uri}&id={id}&kind={kind}



  {{streamMediaLink}}video?uri={uri}&id={id}&kind={kind}



  {{streamMediaLink}}audio?uri={uri}&id={id}&kind={kind}

Parameter Description
uri Content URI for the media, e.g. content://media/external/images/media/26423
id Media id, e.g. 26423.
kind Determine how the media is displayed.
0             
Image - display at the original size.
Video - stream the video.
Audio - stream the audio.
1             
Image - display the micro thumbnail.
Video - display the micro thumbnail.
Audio - display the album art.
2             
Image - display the mini thumbnail.
Video - display the mini thumbnail.
Audio - display the album art.
