vCard API

Send vCard


{{vCardLink}}

Parameter Description
to The destination number or contact name. Mandatory parameter.
cc CC destination number or contact name.
bcc BCC destination number or contact name.
subject vCard message subject.
body vCard message body.
deliveryReport 0 for no delivery report, any other values indicate delivery report is required. Default to 0 for no delivery report.
readReport 0 for no read report, any other values indicate read report is required. Default to 0 for no read report.
vCard vCard content.

Response

{ "vCard": { "bcc": "", "body": "msg body",
"cc": "", "date": "2016-03-20T21:59:45GMT+08:00",
"to": "8632323232323", "subject": "My vCard", "id": "6492044552370211736",
"isDelivered": false, "isRead": false, "readReport": false,
"deliveryReport": false }, "description": "", "requestMethod": "POST",
"isSuccessful": true }

Field Description
vCard vCard object containing the following fields for the sent message.

to
To number.
cc
CC number.
bcc
BCC number
id
A unique identifier that can be used to check the message status.
subject
Message subject
body
Message body
deliveryReport
Flag to indicates if delivery report is required.
readReport
Flag to indicates if read report is required.
isDelivered
Flag to indicates if message is delivered.
isRead
Flag to indicates if message is read.
date
Message sent date.
description Contains the error message if isSuccessful is "false".
requestMethod Request method. Default to "GET", "POST" or "DELETE" depending on the HTTP method.
isSuccessful true if operation is successful. Otherwise returns false.

Note


Generate vCard

Use the Contact API to get the vCard for stored contacts.

Check vCard Message Status

The vCard message is actually a MMS message. Use the MMS API to check the message status.
